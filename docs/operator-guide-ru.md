# Operator guide (RU) — быстрый рабочий цикл

## 1) Первый вход
1. Откройте `http://<ip>:8080`.
2. Проверьте вкладку **Система** (health/версии).
3. На вкладке **Камера** проверьте RTSP и сохраните настройки.

## 2) Работа с ROI
1. На вкладке **Камера** кликайте по кадру, чтобы поставить точки ROI.
2. Минимум 3 точки (можно треугольник/многоугольник).
3. Нажмите **Сохранить**.
4. Убедитесь, что полигон виден поверх кадра после обновления.

## 3) Профили настроек
1. Откройте **Настройки**.
2. Для быстрых режимов нажмите **День** или **Ночь**.
3. Нажмите **Сохранить**, затем **Применить**.
4. Если результат хуже, используйте **Откатить последний профиль**.

## 4) Белый список
1. Откройте **Белый список** и добавьте нужные номера.
2. Можно быстро добавить номер из **Dashboard** или **События**.

## 5) Диагностика
- Вкладка **Диагностика** в Settings: включайте debug параметры только на время расследования.
- После диагностики верните `SAVE_*` и частые логи в безопасный минимум.

## 6) Что делать при проблемах
1. Проверить `/health`.
2. Проверить статус `rtsp_worker` в логах контейнера.
3. Проверить корректность RTSP URL.
4. Проверить, что ROI не перекрывает «пустую» область.
5. При необходимости применить rollback профиля.
